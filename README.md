# Transaction Processing Engine

A high-performance asynchronous transaction processing engine inspired by architectures used by fintechs and modern payment systems.

## Features

- Asynchronous transaction processing
- Concurrent workers
- Scalable architecture
- Transaction lifecycle management
- Fault tolerance and retries
- REST API
- Event-driven architecture

## Tech Stack

- Go
- PostgreSQL
- RabbitMQ
- Docker
- Clean Architecture
